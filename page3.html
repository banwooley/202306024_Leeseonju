유튜브 갤러리

[ Design Tip ] 디자인 팁

디자인을 처음 시작하실 때 꼭 기억해둬야 할 5가지 팁 을 존코바의 경험을 바탕으로
설명한 영상중 첫번째 영상

[ Design Comment ] 디자인 참견러

구독자들의 디자인을 직접 보고 평가하며, 지적할 점 과 좋은점을 같이 집어주는
존코바 유튜브의 대표적 콘텐츠중 하나

[ Photoshop Tutorial ] 포토샵 강좌

존코바가 직접 영상으로 강의하는 포토샵 강좌이며,
기초부터 차근차근 알려주는 것 부터 흥미로운 기능 들을 알려준다

[ Aftereffect Tutorial ]
애프터이펙트 강좌

존코바가 직접 영상으로 강의하는 에프터이팩트 강좌이며,
기초부터 차근차근 알려주는 것 부터 다양한 모션 그래픽 들을 알려준다

[ JohnKOBA Tip ] 존코바 팁

존코바가 직접 디자인과 툴에 관한 팁 들을 알려준다
